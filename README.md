## 4. Collection Steps
### A. Medical Team
1. Export **CGM data** (CSV) ‚Äì meal time + glucose values  
2. Take **food photos** ‚Äì send image + weight (g)  
3. Fill **Google Forms** (link below)

### B. Technical Team
1. Use **LogMeal API** ‚Üí convert image ‚Üí macro (carb/protein/fat)  
2. Run **cleaning script** ‚Üí Thai language + length + duplicate  
3. Export **JSONL** ‚Üí upload to GitHub / GDrive
